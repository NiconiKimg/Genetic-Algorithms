## Configuración del entorno


>[!NOTE]
>
> 1. Instalar el paquete virtualenv
> 
>    ```python
>    pip install virtualenv    
>    ```
>    
> 2. Correr el siguiente comando
>    ```python
>    virtualenv env    
>    ```
> 3. Iniciar el entorno virual 
>    ```python
>    ./env/Scripts/activate   
>    ```
> 4. Instalar requeriments.txt
>    ```python
>    pip install "requeriments.txt"   
>    ```


> [!IMPORTANT]
> 
> Puede suceder que el cmd tire una alerta de ejecución de scripts desactivada.
> 
> 
> En ese caso hay que desactivar la seguridad que bloquea la ejecución de scripts de PowerShell
> 
> Se recomienda ver el siguiente video para solucionarlo
> 
> https://youtu.be/SGsI952Vas8?si=-EYLBsrPt0N41rpb
